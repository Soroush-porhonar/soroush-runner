# Soroush Runner

A browser-based arcade game inspired by the classic **Road Runner**, built with **TypeScript**, **jQuery**, **HTML**, **CSS**, **Vite**, and **npm**.

This project was developed to strengthen my understanding of front-end architecture, game development principles, and interactive browser applications. It includes responsive player controls, collision detection, scoring mechanics, animations, and modular TypeScript code.

## Tech Stack

- TypeScript
- jQuery
- HTML5
- CSS3
- Vite
- npm
